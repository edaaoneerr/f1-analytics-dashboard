# Formula 1 Interactive Data Analysis Dashboard

This repository contains the team project for the **COMP7029: Group Software Project** module.

The aim of the project is to design and implement an **interactive Formula 1 data dashboard** using Python (Dash), supported by statistical analysis and visualisation. The dashboard allows users to explore historical Formula 1 data and gain insights into driver performance, race outcomes, and circuit characteristics.

## Academic Context
This project is developed as part of a **team-based Agile assignment**.  
The team follows Scrum practices with multiple two-week sprints, documented sprint backlogs, meeting minutes, and individual contributions.

The dashboard is built incrementally, starting from data exploration and preprocessing, followed by analysis, modelling, visualisation, and evaluation.

## Dataset
The project uses **Formula 1 Historical Data (1950–2024)** sourced from Kaggle:

https://www.kaggle.com/datasets/rohanrao/formula-1-world-championship-1950-2020

The raw dataset consists of multiple CSV files, which are cleaned, merged, and reduced into a single analytical dataset.

## Technology Stack
- Python
- Dash (Plotly)
- Statistical modelling and data visualisation libraries
